# Horizon Landing Page

Horizon is a premium, competition-ready Web UI/UX landing page piece built with modern web technologies. It focuses on immersive 3D experiences, buttery-smooth scroll animations, and futuristic macro-aesthetics to deliver a high-end feel.

## 🚀 Tech Stack

- **Framework:** [Next.js 15](https://nextjs.org/) (App Directory)
- **Styling:** [Tailwind CSS v4](https://tailwindcss.com/) & Vanilla CSS Modules
- **Animations:** [GSAP](https://gsap.com/) (ScrollTrigger, Timelines)
- **3D Graphics:** [Three.js](https://threejs.org/) (Custom Shaders, Post-Processing Bloom)
- **Smooth Scrolling:** [Lenis](https://lenis.studiofreight.com/)
- **Icons:** [Lucide React](https://lucide.dev/)

## ✨ Key Features & UI Upgrades

- **Interactive 3D Hero:** A fully customized Three.js scene featuring generative starfields, custom shaders (nebulae), and variable-speed parallax scrolling mountains.
- **Premium "Magnetic" Interactions:** The Custom Cursor and Navbar Links use math-driven GSAP properties to mimic magnetic pull effects commonly found in award-winning design agency sites.
- **Organic Motion:** Floating geometric glassmorphic shapes in the CTA area, paired with a constant, generative 'Cosmic Dust' particle system running in the background.
- **Performance Minded:** Scroll animations use GSAP's optimized update ticks; heavy canvas processing includes cleanup and observer hooks.

## 📦 Getting Started

First, install dependencies:
```bash
npm install
```

Then, run the development server:
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) (or whichever port Next starts on) in your browser to see the result.
